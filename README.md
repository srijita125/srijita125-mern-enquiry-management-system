# MERN Enquiry Management System

A full-stack MERN application built using React, Node.js, Express, and MongoDB.

## Features
- Enquiry form submission
- View enquiry list
- Delete enquiries
- REST API integration
- Responsive UI

## Tech Stack
- Frontend: React, Axios, Tailwind / Flowbite
- Backend: Node.js, Express.js
- Database: MongoDB
